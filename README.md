# 🏚️ INTERNET GHOST TOWN

**[RU]** Цифровое кладбище забытых ссылок и перехваченных сигналов. Проект исследует эстетику «мертвого интернета», позволяя пользователям находить и сохранять осколки прошлого.

**[EN]** A digital graveyard of forgotten links and intercepted signals. This project explores the "dead web" aesthetics, allowing users to find and preserve fragments of the past.

---

## 📡 Features / Функции

- **[GLOBAL_BROADCAST]**: Глобальный чат для связи между выжившими в пустоте.
- **[DEEP_SCAN]**: Система глубокого сканирования эфира для поиска случайных артефактов.
- **[MEMORY_LEAK]**: Визуальные эффекты старых ЭЛТ-мониторов и помех.
- **[THE_MORGUE]**: Архив удаленных и забытых страниц.

## 🛠️ Tech Stack / Технологии

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Node.js, Express.
- **Database**: SQLite (via `better-sqlite3`).

---

## 🚀 How to Run / Как запустить

1. **Install dependencies / Установите зависимости**:
   ```bash
   npm install
   ```

2. **Run development server / Запустите сервер разработки**:
   ```bash
   npm run dev
   ```

3. **Build for production / Сборка для продакшена**:
   ```bash
   npm run build
   npm start
   ```
